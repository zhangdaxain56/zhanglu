javaDemo的不知道第几天

发表于 2019-11-27 | 分类于 java

public void chageDemo1(int a){

a=123;

}

public void chageDemo2(Person person){

person.setPersonName(“haha”);

}

public void chageDemo1(String a){

a=”xxxxx”;

}

…..main(String[args]){

int a=30;

chageDemo1(a);

System.out.println(a);

Person person=new Person(“abc”);

chageDemo2(person);

System.out.println(person.getPersonName());

String b=”abc”;

chageDemo2(b);

System.out.println(b);

}

以上输出为 30 ，xxxxx, abc

因为main方法的a调用chageDemo1()，只是复制了a的值赋给了方法中的a，因为mian中的a的作用域只在main中，所以a还是30并没有发生变化

当调用chageDemo2()时 传入的是地址，chageDemo2中的person的指针页指向那个地址，所以改变的是地址中的值，输出是xxxxx

String创建时先会到StringPool中找是否有abc 如果没有就创建一个，有就直接用，当调用chageDemo3时，先会创建一个新的string指向那个新的值，而原来的string还是指向原来的那个pool中的abc
